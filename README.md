# Network-PCAP-Detection
Network Intrusion Detection (PCAP) Project with Wireshark and Splunk

# Overview
This project aims to detect and analyze suspicious network activities using packet capture (PCAP) data. A simulated attack was carried out using Kali Linux, and the resulting network data was captured on an Ubuntu system and analyzed with Wireshark.

# Objectives
- Simulate network-based attacks using Nmap
- Capture live network traffic using tcpdump
- Analyze packet-level data using Wireshark
- Identify scanning behavior and attacker patterns
- Perform traffic analysis and endpoint identification
